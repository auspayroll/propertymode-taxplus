ReadMe

This site will allow you to generate E-PAY numbers into our system. They will be links to registered users and regional sectors.

  • Step 1. - Add a collection group from the above menu. It is up to you how to organise collection groups, but they are simply collections of registered users.
  • Step 2. - Add registered users from the 'Register Collector' link above. Allocate to the Collection group you just created.
  • Step 3. - Add registered users from the 'Register Collector' link above. You can put in a registration number for each user (optional). Allocate to the Collection group you just created from selection list.
  • Step 4. - Select 'Generate EPays' link above. Enter the number of EPay numbers to generate. Then allocate, a collection group and sector. After clicking button, it will generate Epays evenly amongst registered users in the collection group.
  • Step 5. - From the 'Epay log' link above, you will see a batch line for Epays you just created. Click 'Download CSV' for that line.
  • Step 6. - Perform a mail merge in a spreadsheet, eg. OpenOffice. using the CSV downloaded. If the registered user is to be using paper forms, your mail merge form will need to be a complete form so that details can be re-entered in the system later. Otherwise, the mail merge forms will only need to be the receipts that are given out. Ensure that the registered user/operators name and epay number appear on the receipts.
