Business Screen Layout Change - Tuesday 27 January 2015

When coming from 'Submit Taxes' for businesses, you will be directed to a new screen layout to pay taxes for businesses, similar to that of the new property screen layout. The menus that were on the left hand side will now be layed out horizontally near the top of the screen. All the same functionality is still present. This is to make effective use of smaller tablet screen sizes. Also some of the fields that were displayed have been deprecated.

New Payments Screen + Reverse Payments - Tuesday 27 January 2015

From the top menu of both the business and property pages, there is a 'payments' link. This shows payments that are processed in sequential order by date processed. Notice there is a running balance for each business and property. If there is an overpayment in one transaction, it will be carried over (carried forward) into the next transaction. Notice that each Business and Property will have a credit balance displayed at the top of the page. This will be the same as 'carried forward' amount in the last transaction processed.

There is also a 'Reverse Payments' button to undo transaction entries entered, however it is only available to super users. Any reversals made will be logged. Please do not use the old 'reverse payments' page. (I will be removing that link )

Outstanding Invoices - Tuesday 27 January 2015

New Invoices have been generated on the ftp server, for outstanding land leases in Kicukiro District. Only land lease fees after 2011 will be included.

Document uploading problem - Tuesday 27 January 2015

The image uploading still relies on some old code which contained a bug preventing images to be uploaded for property receipts for all users other than super users. I have modified this code and should be working properly. Please let me know if there are still issues.

Fees for 2011 land lease have been extended to apply for 2012, with exception of Karmuna and Mugeyo villages in the Kagaram sector (Residential fees of 70 Rwf/sqm). The latest invoices generates to reflect these changes are on the ftp site ftp://taxplus.co.rw under 12 December 2014 folder.

Change to the property screen layout

When coming from 'Submit Taxes' for properties, you will be directed to a new screen layout to pay taxes for properties. The menus that were on the left hand side will now be layed out horizontally near the top of the screen. All the same functionality is still present. This is to make effective use of smaller tablet screen sizes. Also some of the fields that were displayed have been deprecated.

Additon of new Leasing screens for Land Lease.

Screens with the ability to modify land lease dates have been added. Click 'Leases' from the above property menu. You can also added/remove citizens and businesses on the lease. Also, new leases can be created b selecting 'New Lease' from the property menu.

Note that when leasing dates are changes, this will affected the land lease fee amounts. The fee amounts will get automatically recalculated as soon as the dates are changed.

Update Merge Business function

The merge business function has been updated, so that multiple businesses can be merged if they share a similar business name. Click on 'merge preview' link to view other businesses that have similar names.

Of the businesses that are displayed, selected the ones that you want to merge. Be careful in your selection as not all suggestions presented will be the same business! Check the phone number and tin to determine if it is a match. Once a selection is made, you will be then taken to a screen to select the values of each field taken from each of the selected businesses.
